# Catherine Mutheu Muindi — Portfolio

Personal portfolio website built with HTML, CSS, and JavaScript.

🔗 Live site: https://catherinemuindi.netlify.app

## Built with
- HTML5 / CSS3 / Vanilla JavaScript
- Space Grotesk + Space Mono fonts
- Deployed on Netlify

## Features
- Light / dark mode toggle
- Animated skill bars
- Flip project cards
- Typewriter hero effect
- Scroll reveal animations
- Mobile responsive
